This project is my final disseration peice for my university course. This game is a Roguelite game with custom movement systems and procedural generation.

Gameplay:
  the game is set in a office building, where the player must break as much as possible to gain points.
  each floor is procedurally generated, and the player can buy upgrades that reset every building they go through.

Procedural generation:
  the whole game is preocedurally generated. Each floor number (i.e. building 2 floor 65 or building 10 floor 4) is used as the seed for the generation.
  this means that each floor is always the same, so a player can plan a route through an infinite set of floors.
